Build a modern full-stack mobile-first sports ecosystem application called “Glory Sports” focused on local cricket turf booking, player statistics, team creation, and match management for India.

IMPORTANT:
This is an MVP startup product.
Do NOT overload the application with unnecessary future features.
Focus on:
- smooth user experience
- scalable backend
- clean UI
- fast performance
- real usability

The app should feel like a real startup-ready product, not a demo.

====================================================
MAIN GOAL
====================================================

Create a sports platform where users can:
1. Book sports turf slots
2. Create teams
3. Discover and join unknown players
4. Create and manage matches
5. Track player report cards and ratings
6. Build sports identity through performance statistics

====================================================
ANTI-FAKE SCORE & TRUST SYSTEM
====================================================

This is a VERY IMPORTANT system.

1. Dual Team Match Verification
After match completion:
- Team A captain must confirm score
- Team B captain must confirm score

Only after both confirmations:
- Match becomes verified
- Player statistics update

2. Player Match Confirmation
Players can confirm:
- “Yes, I played this match”

3. Turf-Based Match Validation
Only matches linked to booked/registered turfs can become verified matches.

4. Reputation & Trust Score
Each player should have:
- Fair play score
- Verified match percentage
- Report count

5. Suspicious Match Detection
Flag suspicious matches such as:
- Unrealistic scores
- Too many matches in one day
- Repeated fake activity

Send suspicious matches to Admin review.

6. Verified Badges
System should support:
- Verified Player
- Verified Turf
- Verified Captain
